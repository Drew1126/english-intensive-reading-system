# 考研英语阅读文章生成方案（纯外刊 LLM 改写 + 5500词库硬校验）

> **核心思路**：爬取考研同源外文期刊原文 → LLM 按考研英语一规范改写 → 程序化词库校验 + 超纲词替换 → 合格文章入库。无需真题 RAG，链路极简。

---

## 一、期刊优先级

仅使用以下刊物，其余来源不作为改写素材：

| 期刊 | 优先级 | 适用题材 | 排除内容 |
|------|--------|---------|---------|
| The Economist | 最高 | 社会、科技、经济、公共政策评论 | 纯财报数据罗列 |
| The Guardian | 最高 | 社会民生、教育、环保、文化评论 | 短快讯、娱乐 |
| Scientific American | 最高 | 科普、生物、医学、环境科技 | 硬核实验论文 |
| The Atlantic | 高 | 社会深度、文化、科技评论 | 纯抒情散文 |
| Wired | 中 | AI、互联网、数字社会 | 数码产品测评 |

---

## 二、整体流程

```
原文爬取 & 清洗
      │
      ▼
素材前置筛选（题材 + 文体过滤）
      │
      ▼
LLM 改写（英文 Prompt，输出英文正文）
      │
      ▼
程序化词库校验（对照 5500 词库）
  ├── 超纲词 → LLM 替换循环（最多 2 次）
  └── 仍不通过 → 丢弃，记录原因
      │
      ▼
格式校验（词数 / 段落数 / 句均长度）
  ├── 通过 → 入库
  └── 不通过 → 重写（最多 3 次）
```

---

## 三、素材前置筛选规则

在进入 LLM 改写前，程序化过滤不合格素材：

**保留条件（全部满足）：**
- 题材属于：社会 / 科技 / 经济 / 教育 / 环保
- 文体为：评论、深度分析、观点文章
- 原文词数：800–2000 词（太短信息不足，太长核心论点分散）
- 含明确作者观点或立场

**排除条件（命中任一即排除）：**
- 纯新闻快讯（无分析，只陈述事件）
- 含大量专有数据（股价、选举计票、赛事数据等）
- 纯文学/抒情散文
- 硬核学术实验报告（含大量方法论描述）

---

## 四、改写硬性标准

| 项目 | 标准 | 说明 |
|------|------|------|
| 词数 | 430–520 词 | 程序化校验，含 ±20 词容差 |
| 段落数 | 4–5 段 | 每段 80–120 词 |
| 句均长度 | ≥ 22 词 | 防止改写结果过度碎片化 |
| 长难句数量 | 3–5 句 | ≥ 40 词且含从句嵌套 |
| 超纲词占比 | ≤ 3% | 以 5500 词库为唯一白名单 |
| 让步转折结构 | ≥ 1 处 | while / although / even though / admittedly 等 |
| 输出格式 | 仅英文正文 | 无标题、无作者、无来源、无中文 |

## 五、LLM 改写 Prompt

> ⚠️ **必须使用英文 Prompt**。中文 Prompt 要求英文输出时，模型会用中文逻辑组织内容，导致改写结果带中式英语语感。

### System Prompt

```
You are an expert editor specializing in rewriting English journal articles 
into academic reading passages for the Chinese graduate entrance exam 
(考研英语一 / 英语二).

Your rewrites must match the authentic style, sentence complexity, 
and logical structure of real 考研 reading passages. 
You do NOT generate questions. You output only the rewritten passage.
```

### User Prompt 模板

```
Rewrite the following journal article excerpt into a 考研英语一 
reading comprehension passage. Follow ALL rules below strictly.

## Format Rules
- Total length: 430–520 words
- Paragraphs: 4–5, each 80–120 words
- Output: English body text only — no title, no author, no source, 
  no Chinese, no extra commentary

## Content Rules
1. DELETE: lengthy background setup, excessive examples, 
   specific names of minor figures, institutions, 
   and closing policy recommendations or future outlooks
2. PRESERVE: the core argument, key evidence, 
   and the author's original stance — do not alter the viewpoint
3. REWRITE all sentences in your own words — 
   do not copy phrases verbatim from the source
4. DO NOT fabricate facts, data, or claims not present in the source

## Style Rules
1. Use formal academic English — no colloquialisms, 
   no simplified "plain English" style
2. Sentence complexity: include 3–5 complex sentences (40+ words) 
   using relative clauses, participial phrases, or appositive structures
3. Include at least one concessive structure 
   (e.g., "While researchers agree that..., the evidence suggests...")
4. Paragraph structure: vary between these logical patterns — 
   do NOT force every passage into the same mold:
   - Phenomenon → Cause → Author evaluation
   - Claim → Counterargument → Rebuttal
   - Problem → Research findings → Broader implications
   - Contrast between two perspectives → Author's positioning

## Vocabulary Rules
Use only common academic English vocabulary. 
Avoid rare technical jargon, low-frequency academic terms, 
and discipline-specific terminology.
When a concept requires a specialized term, 
rephrase it using accessible vocabulary.

## Source Article
{{article_text}}
```

## 六、5500 词库校验流程（程序化执行）

> Prompt 里的词汇约束只是引导方向，模型无法真正对照词库。词汇合规必须由程序完成。

### 步骤一：超纲词检测

```python
def detect_out_of_vocab(article_text: str, vocab_set: set) -> list:
    """
    对改写结果分词，检测不在5500词库中的词。
    vocab_set: 已加载的考研5500词库（小写集合）
    返回: 超纲词列表及其位置
    """
    tokens = tokenize(article_text)  # 使用 nltk 或 spacy 分词
    oov_words = []
    for token in tokens:
        lemma = lemmatize(token.lower())  # 还原词形（runs → run）
        if lemma not in vocab_set and not is_proper_noun(token):
            oov_words.append(token)
    return oov_words
```

**词形还原说明**：检测前必须对词库和文章词汇都做词形还原（lemmatization），否则 `running` 会被误判为超纲词。

**专有名词豁免**：人名、地名、机构名不计入超纲词统计（`is_proper_noun` 用 NER 识别）。

### 步骤二：超纲词替换

若超纲词占比 > 3%，调用 LLM 执行定点替换：

```
The following words in the passage are outside the allowed vocabulary list.
Replace each one with a simpler, more common synonym that preserves the meaning.
Do not change any other part of the passage.

Words to replace: {oov_word_list}

Passage:
{article_text}

Output the revised passage only.
```

替换后重新执行步骤一校验，最多循环 2 次。2 次后仍超标 → 丢弃该素材。

### 步骤三：词数 & 段落校验

```python
def validate_format(article_text: str) -> dict:
    word_count = count_words(article_text)
    paragraphs = [p for p in article_text.split('\n\n') if p.strip()]
    avg_sentence_len = calculate_avg_sentence_length(article_text)
    
    return {
        "word_count_ok": 430 <= word_count <= 520,
        "paragraph_count_ok": 4 <= len(paragraphs) <= 5,
        "avg_sentence_len_ok": avg_sentence_len >= 22,
        "word_count": word_count,
        "paragraph_count": len(paragraphs),
        "avg_sentence_len": avg_sentence_len
    }
```

不通过时，将具体校验结果附加到 Prompt 末尾，要求模型针对性修改，最多重试 3 次。

## 七、关键约束汇总

| 类别 | 约束内容 |
|------|---------|
| 词汇 | 以本地5500词库为唯一白名单；超纲词占比 ≤ 3%；专有名词豁免 |
| 格式 | 词数 430–520；段落 4–5；句均长度 ≥ 22 词 |
| 句式 | 长难句 3–5 个；含至少1处让步转折结构 |
| 内容 | 不照搬原文；不篡改作者立场；不捏造事实 |
| 输出 | 仅英文正文；无标题/作者/来源/中文 |
| 题材 | 社会/科技/经济/教育/环保；排除政治敏感、宗教、种族 |
| Prompt | 必须使用英文 Prompt，禁止中文 Prompt 要求英文输出 |
